# YouTube Script Template

Use this format for any YouTube video script. The skill should fill in the bracketed sections.

---

## VIDEO META

- **Working title**: [direct, specific, contains keyword if relevant]
- **Target length**: [2-4 min for explainer, 5-8 min for deep dive]
- **Audience**: [dm2buy seller / Indian D2C founder / agency / general]
- **Goal**: [one of: awareness / waitlist signup / explainer for inbound DMs]
- **Channel**: Mayank's YouTube
- **CTA**: [one only — usually "DM @mayankmalikx 'migrate' for early access" or "Link in description"]

---

## STRUCTURE

### HOOK (0:00 — 0:08)

The first 8 seconds determine retention. Three options that work:

**Option A — The number**
> "I migrated 213 products from a dm2buy store to Shopify in 4 minutes. Here's what's in the tool."

**Option B — The confession**
> "dm2buy doesn't have a CSV export. So I built one."

**Option C — The direct address**
> "If you're on dm2buy and you want to move to Shopify, watch this."

Write the hook as both:
- **Spoken** (what Mayank says out loud)
- **On-screen** (text overlay, 5 words max)

---

### CONTEXT (0:08 — 0:25)

Establish credibility + frame the problem in <20 seconds.

- One sentence on who Mayank is ("I'm building a dm2buy → Shopify migration tool")
- One sentence on the problem ("dm2buy doesn't export — so moving means copy-paste, one product at a time")
- One sentence on what this video answers

**Don't**: long backstory, life story, "subscribe for more before we begin"

---

### MAIN ( ~70% of runtime )

Pick ONE shape per video. Don't mix.

**Shape 1: The demo**
- Walk through actually doing the thing
- Screen recording + voiceover
- Show the input (dm2buy URL), the process, the output (CSV file, products in Shopify)
- Real artifacts. Real numbers. No fake data.

**Shape 2: The teardown**
- "Here's what's actually inside a dm2buy store"
- Browser dev tools, network calls, product structures
- Useful for the technical/builder audience

**Shape 3: The lessons**
- "5 things I learned migrating 12 stores by hand"
- Numbered, specific, with examples
- Useful for the audience-building phase

**Shape 4: The comparison**
- "dm2buy vs Shopify: what changes when you move"
- Side-by-side: checkout, themes, apps, payments, international
- Honest about both sides

---

### CLOSE (last 15-20 seconds)

Three beats:

1. **The summary line**: One sentence restating what they just watched ("So that's how a migration works.")
2. **The honest current state**: "Tool's not live yet — I'm doing migrations manually right now while I finish building it."
3. **The CTA**: "DM @mayankmalikx with 'migrate' if you want one of the early slots."

**Don't end with**: "Smash that like button" / "Don't forget to subscribe" / anything performative.

---

## ON-SCREEN TEXT (overlays)

- Always lowercase or sentence case (no all-caps shouting)
- Font: Geist (bold) or JetBrains Mono (technical)
- Color: white default, Rift Mint (#00E5A0) for the key word/number
- Background: dark with subtle blur, or none
- Max 5 words at a time. Never paragraphs on screen.

Format: timestamp + spoken line + overlay text

Example:
```
0:00 — "I migrated 213 products in 4 minutes."  [overlay: "213 products / 4 minutes"]
0:04 — "Here's what's inside the tool."  [overlay: "what's inside"]
0:08 — "dm2buy doesn't have a CSV export."  [overlay: "no CSV"]
```

## B-ROLL / VISUAL CHECKLIST

What to film/screen-record before scripting:
- [ ] Browser open to a real dm2buy store
- [ ] Terminal running the migration (if showing tool)
- [ ] CSV file opening (Excel/Numbers/Google Sheets)
- [ ] Shopify admin → Products → Import flow
- [ ] Migration report (the .md file output)
- [ ] Face shots: medium close-up, eye-level, plain or branded background

---

## THUMBNAIL BRIEF

For every script, propose 2 thumbnail concepts.

**Style canon**:
- Background: Void (#0A0B0F) with subtle grid
- Accent: Rift Mint (#00E5A0) for one word or number
- Type: Geist Black 900 OR JetBrains Mono
- Face: optional — if used, eye-level, normal expression (no shocked-face)
- NO: arrows pointing at face, red circles around things, "MUST WATCH" overlays

**Examples**:
- Big text: "213 PRODUCTS / 4 MIN" — face left, dashboard screenshot right
- Big text: "DM2BUY → SHOPIFY" with arrow, no face
- Big text: "Here's what's inside" — over screen recording still

## FINAL CHECK before delivering script

- [ ] Hook gets to the point in <8 seconds
- [ ] No exclamation marks anywhere
- [ ] No banned words (revolutionary, seamless, leverage, etc.)
- [ ] No emoji in title or description except the allowed set
- [ ] Building-in-public framing (unless launch mode)
- [ ] One CTA, not three
- [ ] Real numbers, real artifacts, not fake demos
- [ ] Close doesn't beg for subs/likes
