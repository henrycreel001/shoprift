# Post Formats

Non-carousel formats for both Instagram accounts.

---

## @shoprift — Single image / single video post

**Image post**:
- One strong visual (screenshot, dashboard, before/after, single brand-canon graphic)
- Caption pattern:

```
[Hook line — one sentence that answers "what is this"]

[2-3 short lines of detail. Real number if possible.]

[One CTA — "DM 'migrate' for early access." or "Link in bio."]

[3-5 relevant hashtags max — #dm2buy #shopify #d2cindia #indianecommerce]
```

**Example caption** (building-in-public mode):

```
The CSV that comes out of a Shoprift migration.

213 products, every field mapped, ready for Shopify's import flow.

DM "migrate" if you want yours next.

#dm2buy #shopify #d2cindia #indianecommerce
```

## @mayankmalikx — Single image / single video post

**Image post**: usually a screenshot — terminal, code, design, dashboard, a DM (blurred names).

**Caption pattern**:
```
[Action line — what you did/shipped/learned today, lowercase OK]

[1-3 lines of detail. Numbers welcome. Be specific.]

[Optional: 1 line of founder reflection or what's next]

[Soft CTA optional — "DM if you want early access" or nothing]
```

**Example**:
```
shipped the dm2buy scraper today.

213 products extracted in 4 minutes. images downloaded, variants mapped, collections preserved.

next: making it self-serve so sellers don't have to DM me.

if you're on dm2buy and want to see this when it's live, DM "migrate".
```

## Stories (both accounts)

**What works**:
- Quick polls ("Are you on dm2buy or Shopify?")
- Build-in-public glimpses ("here's what I'm fixing today" with a screenshot)
- Mini-explainers (3-5 story slides with a single takeaway each)
- Reposted DMs (with permission, blurred) — "here's what a customer asked today"
- Recent post amplification — "new carousel up, swipe up"

**What doesn't**:
- Random aesthetic stuff with no point
- "Good morning ☕" content
- Anything that doesn't pass the four pillars

**Story format**:
- Plain Void background (#0A0B0F) or branded grid
- One line of text per story slide
- JetBrains Mono for technical/numbers, Geist for headlines

## Reels structure

(Already in `references/channels.md` but repeated here for completeness.)

```
HOOK (0-3 sec)
  Spoken: [first line]
  Overlay: [5 words max]

SCRIPT (15-60 sec)
  [one idea, told in short sentences]

ON-SCREEN TEXT
  [reinforcement, not repetition]

B-ROLL
  [what visuals support each beat]

CTA (last 3 sec)
  Spoken: [one CTA]
  Overlay: [DM @handle]

CAPTION
  [1-2 sentences in matching voice + 3-5 hashtags]
```

**Reel types that work for Shoprift**:
1. **The screen-record demo** — no face, just doing the migration on screen, voiceover
2. **The face explainer** — Mayank on camera, 30-second answer to "what is Shoprift"
3. **The build-in-public** — Mayank narrating "what I shipped today" with screen cuts
4. **The teardown** — opening dev tools on dm2buy, showing the structure, explaining why migration is hard

**Reel types to avoid**:
- Trending audio with text overlay only (lazy, doesn't fit voice)
- Tutorial-style listicles with stock footage
- Anything that needs "WAIT FOR IT" type retention bait
